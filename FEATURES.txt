LUCID - Escrow Transaction Management App
==========================================
Feature Log - All Implementations To Date
==========================================

PROJECT OVERVIEW
----------------
Lucid is a React Native (Expo SDK 55) escrow/transaction management app
connected to a Go backend at wizerconsulting.com:9091/api/v1.
Built with custom Satoshi font, 33 custom SVG components, and 5 screens.

-------

   - 4 horizontal scrollable stat cards (All, Action Required, Open, Closed)
   - Transaction list with search and tab filtering (All, Staging, Signed, Disbursed, Closed)
   - Transaction cards showing type, description, amount, role, counterparties, closing date countdown
   - Color-coded status badges

API INTEGRATION
---------------

Transaction Creation (POST /transactions):
- Multipart/form-data format
- Go gin binding dot notation: counterparties[0].email (not counterparties[0][email])
- All counterparty fields sent per API spec (email, name, role, phone, permissions, etc.)
- Permissions sent as counterparties[i].permissions[pi].permission / .value
- Settlements use amount_type field (fixed/percentage)
- Due From / Due To send counterparty email addresses
- Closing date in RFC3339 format (no .000Z, just Z)
- Additional fields: base_currency, is_currency_exchange, exchange_rate, platform_markup

Transaction Listing (GET /transactions):
- Fetches paginated list with status filtering
- Maps backend stages to UI tabs

Transaction Detail (GET /transactions/{id}):
- Case-insensitive email matching for viewer's role
- Renders counterparties, settlements, documents from response
